# UV - Portal Académico de Ingeniería

Sitio web académico desarrollado para organizar y compartir materiales de cursos de ingeniería impartidos por **Prof. Fili**.

## 🌐 Sitio web

Accede al portal en:

👉 https://fili10245.github.io/uv/

---

## 📚 Cursos disponibles

El sitio incluye los siguientes cursos:

### 1. Fundamentos de Transferencia de Momentum
- Conceptos de esfuerzo y viscosidad
- Ecuaciones de transporte
- Balance de cantidad de movimiento

### 2. Mecánica de Fluidos
- Propiedades de los fluidos
- Estática de fluidos
- Análisis de presión y fuerzas

### 3. Métodos Numéricos
- Métodos para encontrar raíces
- Sistemas de ecuaciones
- Técnicas computacionales

---

## 🧩 Estructura del sitio

El sitio está organizado por semanas dentro de cada curso:

- **Semana 1**: Introducción y conceptos básicos  
- **Semana 2**: Desarrollo teórico y aplicaciones  

Cada sección incluye:

- Notas (.md)
- Material en PDF (en desarrollo)
- Tareas y ejercicios
